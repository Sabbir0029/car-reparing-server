# Car Repairing Station Management System

A full-stack web application for managing a car repairing station efficiently.  
This system helps manage customers, vehicles, appointments, repairs, billing, inventory, and reports.

---

# Features

- User Authentication (JWT)
- Role-Based Access Control
- Customer Management
- Vehicle Management
- Appointment Scheduling
- Repair Status Tracking
- Invoice Generation
- Inventory Management
- Reports & Analytics
- Responsive Dashboard UI

---

# Technology Stack

## Frontend
- React.js
- Next.js
- Tailwind CSS

## Backend
- Node.js
- Express.js

## Database
- MongoDB

## Authentication
- JWT (JSON Web Token)
- bcrypt password hashing
